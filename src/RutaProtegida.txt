
El archivo ProtectedRoute.js o RutaProtegida
es un componente de React que sirve para proteger rutas que solo deberían ser accesibles 
si el usuario ha iniciado sesión correctamente.

const token = localStorage.getItem("token"); Esta instrucción revisa si existe el token
return token ? children : <Navigate to="/" replace />; Si no hay token, redirige al login

En si su funcionamiento es:
1. Verificar si existe un token en localStorage
2. En caso que si existe, renderiza el componente que se envuelve por ejemplo <Navbar /> para /menu
3. En caso que no existe, redirige automaticamente al login "/"
